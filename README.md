# GOVANGO

### Run docker-compose
```bash
docker-compose up -d
```

### Stop docker-compose
```bash
docker-compose down
```

### Stop docker-compose with full data
```bash
docker-compose down -v
```

### PM 2 commands

```bash
pm2 start ./dist/backend/main
```
